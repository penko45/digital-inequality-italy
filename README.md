# digital-inequality-italy
SSDD portfolio project analyzing digital inequality in Italy using Python and Eurostat data
# Digital Inequality in Italy 🇮🇹

## Project Overview
This project analyzes digital inequality and socio-economic disparities across Italian regions using Eurostat and ISTAT datasets.

It focuses on:
- Unemployment rates
- Digital access inequality
- Education and regional differences

## Tools Used
- Python (Pandas, NumPy)
- Matplotlib / Seaborn
- Jupyter Notebook

## Key Insights
- Southern Italy shows higher unemployment rates
- Digital access is significantly lower in rural regions
- Education level strongly correlates with employment rate

## Datasets
- Eurostat regional unemployment data
- ISTAT digital access statistics
